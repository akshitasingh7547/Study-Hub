# Study Hub 2 🧙‍♀️

A **personal education operating system** designed for serious students. Study Hub 2 combines academics, skills development, career planning, and personal finance into one integrated platform.

## 🌟 Features

### 📚 Core Study Features
- **Dashboard** — Today's schedule, marks overview, motivational quotes, study progress
- **Subject Index** — All 12th board subjects (Physics, Chemistry, Biology, Math, English, IT) with chapter organization
- **Study Hall** — Pomodoro timer, to-do list tracking, ambient music, focus mode
- **Exam Hall** — Test score tracking, customizable timer, multiple exam categories
- **Planner** — Calendar with event management, timetable, reminders

### 🚀 Skills & Career Development
- **Skills Lab** — 5 major skill tracks:
  - 💻 **Coding** (Python, HTML, CSS, JavaScript, SQL)
  - 📈 **Share Market** (Quant Analyst pathway)
  - 🎨 **Digital Editing** (Posters, logos, thumbnails)
  - 🤖 **AI Engineering** (ML, Deep Learning, OpenAI APIs)
  - ✍️ **Writing Skills** (Essays, blogs, debates)
- **Career Vault** — Explore career paths, create roadmaps, track interest levels

### 📊 Progress & Analytics
- **Analytics Dashboard** — Study hours, subject accuracy, test scores, weak chapters
- **Achievement Hall** — Streaks, XP, milestones
- **Library** — Digital book tracking with reading progress

---

## 🛠️ Tech Stack

- **Frontend:** React.js + Tailwind CSS
- **Styling:** Custom Slytherin theme with smooth animations
- **State Management:** React Hooks
- **Routing:** React Router v6
- **Icons:** Lucide React
- **Responsiveness:** Mobile-first design

## 🚀 Getting Started

### Prerequisites
- Node.js (v14+)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/akshitasingh7547/Study-Hub-2.git
   cd Study-Hub-2
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```
